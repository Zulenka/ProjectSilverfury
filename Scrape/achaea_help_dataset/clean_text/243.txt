4.11.3 CONFIG NEWSIGNORE - ignoring certain news sections
You may choose to
ignore
certain
news
sections. The
command
is
CONFIG
NEWSIGNORE
. Just type that alone to see command options, also shown below:
CONFIG NEWSIGNORE ADD|
REMOVE
group
(s)
CONFIG NEWSIGNORE NONE
examples:
CONFIG NEWSIGNORE ADD MEETS POETRY (now you ignore meets and poetry)
CONFIG NEWSIGNORE REMOVE MEETS     (now
ignoring
only poetry)
CONFIG NEWSIGNORE NONE             (ignore
nothing
: see
everything
!)
The
groups
you may list in newsignore are PUBLIC, COMMENTS, CHANGES, and POETRY.
