19.13.3 Class Customisation
(See also HELP CUSTOMISATION REQUESTS and
HELP
CUSTOMISATION
GUIDELINES)
Class
Loyals
------------
Occultist
entities may not be customised. Class-specific non-creatures such as summoned maces are not customisable.
Subterfuge
snakes,
Discipline
falcons,
Valour
eagles
,
Dominion
hounds,
Malignity
hyenas,
Grove
golems, Angel, Baalzadeen, Waterweird, Crypt Golems, Aklan and Kindler Servitors can be customised.
Class Loyals: Procedure
-----------------------
You can use 200
bound
or customisation
credits
to customise a class loyal via the customisation
system
(see
HELP CUSTOMISATIONS
and HELP CUSTOMISATION REQUESTS).
For class loyals that are snakes, falcons, eagles, hounds, or hyenas, you can customise the
denizen
directly. Bear in mind that if you customise a class loyal that is not collared with a Collar of
Lupus
, any customisation will be
lost
when your class loyal dies. These class loyal
customisations
are also not retained through the
quitting
of the class, even if you regain the class again later.
CUSTOMISATION NEW FOR <denizen#>
For class loyals that are angels, baalzadeens, waterweirds, crypt golems, grove golems, aklan servitors, or kindler servitors, use the
following
in the customisation system to create a classloyal customisation:
CUSTOMISATION NEW FOR <loyaltype> CLASSLOYAL
Loyaltype is angel, baalzadeen, waterweird, cryptgolem, grovegolem, aklan, or kindler.
Customisation of these does not require that the loyal be preserved in anyway, and if your angel, golem, weird, etc are killed, the customisation will be retained on resummon. This customisation will also persist through class change.
Customisation of a class loyal must make
sense
for the class and any factionality associated with the class/loyal.
Defiled, fallen, or despoiled angels, or angelic baalzadeens, and similar will not be permitted.
Class
Description
- Viridian
------------------------------
For those with the
Sylvan
class, the viridian form description can now be customised.
This costs 200 bound or customisation credits.
CUSTOMISATION NEW FOR VIRIDIAN will create a customisation request.
If you already have a
custom
viridian description, the new customisation will by default be populated by these existing values. You can then update these or change entirely.
Armour
Customisation
-----------------------
Armour may be customised via the in-game customisation system once it has been
made non-
decay
and resetting (see HELP
ARTEFACT
ARMOUR).
