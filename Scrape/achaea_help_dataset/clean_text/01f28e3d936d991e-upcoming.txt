18.2.4 UPCOMING
UPCOMING
is a quick and easy reference for upcoming
events
. The administration and
organisational leaders can add events with their
time
and further information
in the
following
manner:
UPCOMING
- View a list of upcoming events
UPCOMING INFO <event>
- Get further details of an
event
UPCOMING NEW <org> <name>
- Create a new event
UPCOMING <#> SET INFO
- Add additional information to an event
UPCOMING <#> SET
NAME
<name>
- Change the event's name
UPCOMING <#> SET TIME <yyyy/mm/dd hh:mm:ss>
- Set the time the event occurs at
UPCOMING <#> PUBLISH
- Publish the event for people to see
UPCOMING <#> FINISH
-
Remove
the event from the listing
Discussing UPCOMING listings in the context of the 'upcoming calendar' or 'upcoming events' is completely fine.
However, please be aware that some
admin
-posted listings may be innately
OOC
in
nature
- while these may occasionally be marked with an [OOC] tag for ease of reference, some will not.
Discretion is advised here: avoid fortune telling where possible (i.e a coming story event chapter is likely posted for OOC convenience), and err on the side of caution if you're unsure.
